# SeverinoBiu

Website da SeverinoBiu - Corporação de Ofício de AI

## Sobre o Projeto

Site institucional com estética armorial inspirada na literatura de cordel brasileira. O design celebra a fusão entre tradição artesanal nordestina e tecnologia de inteligência artificial.

## Características

- **Design Armorial**: Estética inspirada no Movimento Armorial e literatura de cordel
- **Paleta de Cores**: Terracota, ocre, azul noturno - cores que remetem à xilogravura
- **Tipografia**: Fontes que evocam o estilo dos folhetos de cordel
- **Responsivo**: Adaptado para todos os dispositivos
- **Animações Suaves**: Interações discretas que valorizam o conteúdo

## Estrutura

```
/SeverinoBiu
  ├── index.html      # Estrutura principal do site
  ├── styles.css      # Estilos com tema armorial
  ├── script.js       # Interatividade e animações
  ├── img/            # Galeria de imagens
  └── README.md       # Este arquivo
```

## Como Visualizar

Abra o arquivo `index.html` em qualquer navegador moderno.

## Tecnologias

- HTML5 semântico
- CSS3 com variáveis customizadas
- JavaScript vanilla para interatividade
- Google Fonts (Smokum, Sirin Stencil, Special Elite)

## Conceito Visual

O design mescla elementos da cultura popular nordestina com a modernidade da AI:
- **Xilogravura Digital**: Bordas marcadas e contrastes fortes
- **Cordel Contemporâneo**: Tipografia que honra a tradição dos folhetos
- **Aquedutos e Asas**: Metáforas visuais de conexão e transformação
